# WAVETYPE_V1
WAVETYPE is a tool to create patterns out of typography
<h1> WAVETYPE </h1> 
				<small> Version 1 </small> <br>
				<small> &#xA9; <a href= "https://judithhoefliger.cargo.site/" target="_blank">judithhoefliger.cargo.site</a></small> <br>  <p> 
				<p>  with this tool you can make some nice patterns out of typography  <p>
				<p> <h2> some quick Informations </h2> <p> 
				- you always need to select your text in order to make changes to it, also if you want to use the random button <br> 
				- you can select it when you klick on the first letter on the word <br>  
				- as a said info, you need a really high number in order to change the direction(angle input) <br>
				- just to say, unfortunately you can't export your visuals as a gift or video because the quality of the export will be really bad, try to do a screen recording, hopefully I'll find a solution for it in a future version  <br> 
				- and be careful if you do a lot of long rows -> performance issue incoming <p>
				<p> now have fun making some patters! <p>
				<h2> tools used </h2>
				<p> <a href="https://seb-prjcts-be.github.io/p5.waves/index.html" target="_blank">p5.Waves</a>, movements for text
