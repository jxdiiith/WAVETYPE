<h1> WAVETYPE </h1> 
				<small> Version 1 </small> <br>
				<small> &#xA9; <a href= "https://judithhoefliger.cargo.site/" target="_blank">judithhoefliger.cargo.site</a></small> <br>  <p> 
				<p> WAVETYPE is a browser-based tool for creating patterns out of typography. With arrangements, repetition, length, wave motion and a bunch of other parameters you can create your own unique visuals.<p>
				<p> <h2> some quick Informations </h2> <p> 
				- you always need to select your text in order to make changes to it, also if you want to use the random button <br> 
				- you can select your texts when you klick on the first letter of the word <br>  
				- as a side info, you need a really high number in order to change the direction(angle input) <br>
				- just to say, unfortunately you can't export your visuals as a gift or video (yet) because the quality of the export will be really bad, try to do a screen recording, hopefully I'll find a solution for it in a future version  <br> 
				- and be careful if you do a lot of long rows -> performance issue incoming <p>
				<p> now have fun making some patterns! <p>
				<h2> tools used </h2>
				<p> <a href="https://seb-prjcts-be.github.io/p5.waves/index.html" target="_blank">p5.Waves</a>, movements for text
